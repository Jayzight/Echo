This is a more simplistic adaption of the classic game, Simon. The rules are simple. A series of tones and lights called echoes will be created. The player must repeat the sequence. If the player succeeds, the series becomes progressively longer and more complex. Once the player fails or the time limit runs out, the game is over.


Created by Jayson Tian in Grade 9. I used some resources to help with the JavaScript code. Linked in the Github Pages.
